**QUICK SETUP**
- Pour installer, éxécuter le script setup.py
- Pour lancer, lancer RIKAgroq pour utiliser l'assistant

**CAPABILITIES**
- Obtenir la localisation
- Capture automatique de l'écran et analyse
- Capture automatique de la webcam et analyse
- Ouvrir une application dans la liste donnée
- Envoyer un courriel
- Éxécuter des actions custom via des protocols
- Sauvegarder un fichier et l'ouvrir
- Lire des courriels
- Reconnaitre une musique
- Recherche sur le web
- Ouvrir un lien web

**OPTIONS**

Certains paramètres peuvent être modifié dans `settings.json`
L'agent peut détecter tout seul si on lui parle ou pas
Overlay hi-tech
Pour lui parler, appeler son nom (Rika par défaut) pour une conversation orale ou faire le raccourcis clavier désigné pour une conversation écrite (par défaut ctrl+alt+r)
_NOTE:_ L'input au clavier ne fonctionnera pas dans d'autres applications si le mode conversation est activé
pour désactiver Rika, simplement lui dire au revoir ou lui demander de se mettre en veille

*TROUBLESHOOTING*

M'envoyer les fichiers log.log et debug.json en me mentionnant l'erreur
